## 문제

[문제 링크](https://boj.kr/11050)

## 입력

<p>첫째 줄에 \(N\)과 \(K\)가 주어진다. (1 ≤ \(N\) ≤ 10, 0 ≤ \(K\) ≤ \(N\))</p>

## 출력

<p>&nbsp;\(\binom{N}{K}\)를&nbsp;출력한다.</p>
